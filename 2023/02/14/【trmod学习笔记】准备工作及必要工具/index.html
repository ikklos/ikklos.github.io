【trmod学习笔记】准备工作及必要工具

发表于 2023-02-14 更新于 2023-08-01 分类于 TRmod学习笔记

前言

没错，我们又从头开始了，从本篇开始我的mod学习主要依赖于tml官方网站的教程和资源而不是国内其他大佬的中文教程，从头开始的同时也补充一些前面的笔记。

语法准备

开发泰拉瑞亚模组不需要太多的编程知识（至少他们是这么说的），但需要懂得C#的语法，官网内给出了一个学习网站 C# Syntax Tutorial ，只不过是全英文的，国内也有较为方便的网站比如 菜鸟教程 ，b站等等，但这些可能缺少一些C#的新特性补充，在前段时间的学习中我发现mod里用了一些C#的新特性，如果需要学习也可以选择购买书籍。

文本编辑器

文本编辑器有很多，注意这些并不是IDE

  • Notepad++
  • Sublime Text
  • Vim
  • Atom
  • Visual Studio Code
  • GNU Emacs (and its iterations)

很明显我们只需要选择其中一个就足够了，我使用的是vscode但是似乎有时候这东西并没有Notepad++用起来顺手，还是看个人习惯。建议都试试。不过它们都比电脑自带的记事本功能更加强大。

IDE（集成开发环境）

比较方便的就是vs了，在组织项目上面它的功能更多也更强大，但正因为功能多，使用者可能会感到迷惑，一些莫名奇妙的标记也会让自己血压升高（ 它老觉得我代码写得难看 ）。但无疑我们需要这样一个集成开发环境，熟练使用之后可以让我们更快找到问题所在。

图形编辑工具（画图工具）

我们需要使用能够创作像素画的工具，这里我选择的是Aseprite，在steam上就可以买到，也可以去github找到开源版本自行编译安装，可惜过程过于繁琐我才懒得弄。

以上，准备工作就基本完成，接下来就可以正式开始创作模组了。
