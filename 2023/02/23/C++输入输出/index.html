输入流&输出流

流

一个流就是一个字符序列，从外界接收输入就称作 输入流 ，向外界输出就是 输出流 ，流 的概念是一种 顺序 的概念，它通常表示数据是顺序输入或输出的

iostream库

C++可以使用iostream库来控制流，它其中有两个基础类型，即istream 和 ostream 。对应着C语言中的标准输入输出函数，iostream库中有cin和cout对象，另外，还有 cerr 和 clog ，cerr 用来输出 警告和错误信息 ，clog用于输出一般性信息 ，可以用来写入日志文件。而在C语言中，其中的stderr和stdin，stdout则是使用文件指针的形式，可以看出C语言与C++的区别。

在上面的代码中，我们见到了两种新的运算符 “>>”和“<<”它们分别是输入、输出运算符，运算符与表达式构成的仍然是表达式，也就有计算结果，以“<<”输出运算符为例，它的左侧只能是ostream类型， **计算结果** 是左侧的值， **副作用** 是将右侧的数据写入左侧的流中。那么std::cout << a << b; 计算顺序如下 (std::cout << a) << b; $\rightarrow$ std::out << b; 当然这只是我们的分析顺序，具体的编译会将表达式进行处理。“>>”输入运算符也有相似的用法，它的左侧只能是istream类型。

缓冲方式一般分为三种：无缓冲、行缓冲和全缓冲，在C语言中常见的输入函数scanf就是采用行缓冲。C语言中，对于缓冲区的约定是：遇到换行符'\n'以及遇到 scanf() 函数等需要输入的时候，或缓冲区已满时，自动将缓冲区的数据刷到设备上生成真正输出。当然C语言也有控制缓冲区的方式。现在我们只关心C++的常用输出方式，即cout，在cout中我们要使用一些操纵符来使输出流做出特定动作从而实现缓冲区的刷新以及将输出打印到屏幕上实现真正输出。
